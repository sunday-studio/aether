# aether structure

press record -> record speech -> speech turns into text -> text sent to openai -> openai returns activity -> activity is stored on jazz

### supported activities:

- gym
- reading
- cooking
- list
- ideation
- weight
- food tracking

### supported actions

- add
- remove
- mark as done
- mark as not done
- edit
- delete

## views;

- today's view
- calendar view with heatmap tracking
- activities views
