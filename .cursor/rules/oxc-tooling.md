# OXC Tooling

Use OXC for frontend formatting and linting in `desktop/`.

- Run `npm run lint` from `desktop/` for lint checks.
- Run `npm run format` from `desktop/` for formatting.
- Run `npm run check` from `desktop/` before frontend commits.
- Do not add ESLint, Prettier, or Biome configs unless the team intentionally reverses this decision.
